# chess

本项目是一个基于Vue的下棋小项目，在九方格子内部的先连成一条直线为胜利者

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```
